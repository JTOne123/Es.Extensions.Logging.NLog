# Es.Extensions.Logging.NLog

NLog provider for Microsoft.Extensions.Logging; .NET Core and ASP.NET Core.

## 如何使用
---
Add dependency in project.json 
```xml
"dependencies": {
   "Es.Extensions.Logging.NLog": "1.0.0-*"
 }
```
